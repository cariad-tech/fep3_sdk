==================
FEP System Library
==================

The :term:`FEP System Library` provides a standalone solution to control,
monitor and configure a :term:`FEP System` over the :ref:`label_service_bus`.
This library does not depend on the :ref:`FEP_Participant_Library`.

The following chapters provide documentation for the :term:`FEP System Library` . For further information regarding how 
to use the :term:`FEP System Library` to control a :term:`FEP System` have a look at the :ref:`User Documentation <FEP_System_Library>`. 
For a deep dive into the inner working of the library have a look at the :ref:`Developer Documentation  <FEP_System_Dev_doc>`.
